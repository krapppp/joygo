# 📑 파이썬프로그래밍 3주차(3장) 통합 요약 노트

## 1. 리스트 (List)
여러 개의 값을 하나의 변수에 함께 모아 저장할 수 있는 자료형.
- **기본 구조:** 대괄호 `[]`를 사용하며, 각 값은 쉼표(`,`)로 구분함. (예: `[값1, 값2, 값3]`)

## 2. 리스트 조작 메서드
- **값 추가 (`append`):** 리스트의 맨 마지막에 새로운 값을 추가함.
  - 사용법: `리스트명.append(추가할_값)`
- **정렬 (`sort`):** 리스트의 값을 오름차순(알파벳, 한글 자모, 숫자 크기 순)으로 정렬함.
  - 사용법: `리스트명.sort()`
- **개수 세기 (`count`):** 리스트 안에 특정 값이 몇 개 들어 있는지 셈.
  - 사용법: `리스트명.count(찾을_값)`

## 3. 인덱싱과 슬라이싱
### 인덱싱 (Indexing)
- 리스트 내의 특정 값에 접근하기 위해 주소 번호(Index)를 사용하는 것.
- 파이썬의 인덱스는 **0부터 시작**함. (첫 번째 요소는 `[0]`)
- 사용법: `리스트명[인덱스_번호]`
- 주의: 리스트의 크기(길이)를 벗어나는 인덱스에 접근하려 하면 에러가 발생함.
- **값 제거 (`del`):** `del 리스트명[제거할_인덱스]`를 통해 특정 위치의 값을 지울 수 있음.

### 슬라이싱 (Slicing)
- 리스트에서 여러 개의 값을 한 번에 잘라서 가져오는 것.
- 사용법: `리스트명[시작_인덱스 : 끝_인덱스+1]`
  - *참고:* 지정한 범위에서 '끝_인덱스'의 바로 앞까지만 가져옴.
- 특징: 슬라이싱을 해도 원본 리스트의 내용은 변하지 않음.
