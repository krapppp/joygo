# 📑 파이썬프로그래밍 2주차(2장) 통합 요약 노트

## 1. 자료형과 연산자
### 자료(Data)와 자료형(Data type)
프로그램의 기본 재료인 자료를 종류에 따라 구분한 것.
- **숫자형:** 정수(integer), 실수(float) 등으로 이루어진 자료.
- **문자열(String):** 문자를 나열한 자료. 작은따옴표(`'`)나 큰따옴표(`"`)로 감싸서 표현함.

### 연산자 (Operator)
- `+` (더하기), `-` (빼기), `*` (곱하기), `/` (나누기 - 결과는 항상 실수형)
- `**` (거듭제곱)
- `//` (몫), `%` (나머지)

## 2. 문자열 연산
문자열도 숫자처럼 연산 기호를 사용하여 조작할 수 있음.
- **문자열 더하기 (`+`):** 여러 개의 문자열을 하나로 연결함 (예: `"Hello" + "World"`).
- **문자열 곱하기 (`*`):** 문자열을 주어진 정수 횟수만큼 반복함 (예: `"Python" * 3`).

## 3. 변수 (Variable)
값을 저장하는 공간. 숫자뿐 아니라 문자열 등 다양한 자료형을 저장할 수 있으며, 언제든지 값을 변경할 수 있음.
- **할당 연산자 (`=`):** 오른쪽의 값을 왼쪽의 변수에 저장함 (예: `변수명 = 값`).
- **변수 이름 짓기 규칙:**
  - 영문자, 숫자, 밑줄(`_`) 사용 가능.
  - 대소문자를 엄격히 구분함.
  - **불가:** 숫자로 시작할 수 없음, 공백 포함 불가, 파이썬 예약어(예: `if`, `for`) 사용 불가.

## 4. 주석 (Comment)
사람을 위해 코드에 남기는 설명글.
- `#` 기호를 사용함. `#` 뒤에 적힌 문장은 컴퓨터가 처리하지 않고 무시함.
- 특정 코드를 잠시 실행되지 않게 가려둘 때도 유용함.
