# 📑 파이썬프로그래밍 5주차(5장) 통합 요약 노트

## 1. 참과 거짓 (Boolean)
파이썬에서 논리적인 참과 거짓을 표현하는 자료형입니다.
- 값은 오직 두 가지: `True`(참) 또는 `False`(거짓).
- *주의:* 반드시 첫 글자를 대문자로 적어야 함.

## 2. 비교 연산자와 논리 연산자
### 비교 연산자
두 값을 비교하여 그 결과로 `True` 또는 `False`를 반환합니다.
- `<` (작다), `>` (크다)
- `<=` (작거나 같다), `>=` (크거나 같다)
- `==` (같다), `!=` (다르다)

### 논리 연산자
여러 개의 조건식을 하나로 묶어 판단할 때 사용합니다.
- **`and` (그리고):** 양쪽 조건이 **모두** 참일 때만 `True`를 반환. (하나라도 거짓이면 `False`)
- **`or` (또는):** 양쪽 조건 중 **하나라도** 참이면 `True`를 반환. (모두 거짓일 때만 `False`)
- **`not` (부정):** 현재 논리값을 뒤집음. `True`는 `False`로, `False`는 `True`로 바꿈.

---

## 3. 조건문 (if, elif, else)
특정 조건이 참인지 거짓인지 판단하여 프로그램의 실행 흐름을 제어합니다.

### 기본 구조
```python
if 조건식1:
    # 조건식1이 True일 때 실행할 코드블록
elif 조건식2:
    # 조건식1이 False이고, 조건식2가 True일 때 실행할 코드블록
else:
    # 위의 모든 조건이 False일 때 최종적으로 실행할 코드블록
```

### 특징
- `if`문 다음에는 반드시 콜론(`:`)을 찍고, 그 아래 실행할 코드는 들여쓰기(Indentation)를 해야 합니다.
- `elif`는 'else if'의 줄임말로 여러 개의 조건을 순차적으로 판단할 때 사용하며, 필요한 만큼 여러 개 쓸 수 있습니다.
- `else`는 조건식 없이 사용되며, 모든 조건이 맞지 않을 때의 예외 처리를 담당합니다.
- `elif`와 `else`는 단독으로 쓰일 수 없으며 항상 `if`와 함께 사용해야 합니다.
