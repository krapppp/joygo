# 📑 파이썬프로그래밍 1주차(1장) 통합 요약 노트

## 1. 프로그래밍의 이해
- **프로그래밍 언어:** 컴퓨터가 이해할 수 있는 언어. 컴퓨터에게 일을 시키기 위해 사용함.
- **주요 용어:**
  - **코드(Code):** 프로그래밍 언어로 작성한 명령 한 줄 한 줄.
  - **코딩(Coding):** 코드를 작성하는 과정.
  - **프로그램(Program):** 서로 관련 있는 코드들을 모아 놓은 작업 지시서.
  - **프로그래밍(Programming):** 언어를 이용해 논리적이고 명확하게 프로그램을 만들어가는 과정 전체.

## 2. 파이썬(Python)의 특징
- **인터프리터 언어:** 코드를 한 줄씩 읽어가며 바로 통역(실행)하여 초보자가 배우기 좋음.
- **장점:** 문법이 간결하고 쉬움, 생산성이 뛰어남, 오픈소스이면서 강력함.
- **활용 분야:** 데이터 과학(빅데이터 분석), 인공지능(딥러닝), 웹 개발(Django) 등. (구글, 인스타그램, 나사 등에서 사용)

## 3. 개발 환경 및 기초 문법
- **IDLE (Integrated Development and Learning Environment):** 파이썬의 기본 개발 환경.
- **대화형 셸 (Interactive Shell):** `>>>` 프롬프트에 파이썬 명령어를 한 줄씩 입력하여 즉시 실행 결과를 확인할 수 있는 창.
- **기초 출력:** `print("Hello World!")`를 사용하여 문자열을 화면에 출력.

## 4. 터틀 그래픽 (Turtle Graphics)
- 화면에 거북이를 띄워 이동시키면서 그림을 그리는 시각적 모듈.
- **주요 함수:**
  - `import turtle`: 터틀 모듈 불러오기.
  - `t.forward(100)`: 100픽셀 전진.
  - `t.left(90) / t.right(90)`: 왼쪽/오른쪽으로 90도 회전.
  - `t.color('색상')`: 펜 색상 변경.
