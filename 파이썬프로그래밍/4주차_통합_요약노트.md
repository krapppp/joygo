# 📑 파이썬프로그래밍 4주차(4장) 통합 요약 노트

## 1. 반복 작업과 IDLE 에디터
### 반복문 개요
- **for문:** 반복 횟수가 명확하게 정해져 있을 때 주로 사용.
- **while문:** 특정 조건이 만족되는 동안 계속 반복할 때 주로 사용.

### IDLE 에디터 사용법
- 한 줄씩 실행되는 대화형 셸과 달리, **여러 줄의 코드를 한 번에 작성하고 묶어서 실행**할 수 있는 창.
- 실행 과정: `[File]` - `[New File]`로 창을 열어 코드 작성 -> `[File]` - `[Save]`로 파일(.py) 저장 -> `[Run]` - `[Run Module]` (단축키 F5)로 코드 전체 실행.

## 2. for문의 구조와 들여쓰기
### for문 기본 구조
리스트(또는 문자열, range)의 각 값을 순서대로 하나씩 꺼내와 변수에 담은 후 코드블록을 반복 실행함.
```python
for 변수 in 리스트_또는_문자열:
    실행할_명령 (코드블록)
```
- 예: `for i in [1, 2, 3]: print(i)`
- `in` 뒤에 올 수 있는 것: 리스트, 문자열(각 문자를 하나씩 가져옴), `range()` 함수 등.

### 코드블록과 들여쓰기 (Indentation)
- **들여쓰기:** 반복해서 실행될 명령(코드블록)을 묶어주는 역할.
- 파이썬에서는 들여쓰기가 문법적으로 매우 중요함. `for`문 끝에 콜론(`:`)을 찍은 후 다음 줄부터 반드시 들여쓰기를 해야 함.
- **규칙:** 보통 **띄어쓰기 4칸** 혹은 **Tab 키**를 사용하며, 하나의 프로그램 안에서는 반드시 한 가지 방식만 통일해서 사용해야 에러가 발생하지 않음.
