# 📑 정보시스템보안 4주차 통합 요약 노트

## 1. 파일 삭제 및 링크 (3장 심화)
### 파일 삭제 명령 (`rm`)
- **기본 사용:** `rm [파일명]` (파일을 즉시 삭제하며 복구 불가).
- **옵션:**
  - `-i`: 삭제 전 확인 메시지를 출력하여 실수를 방지함.
  - `-r`: 디렉터리를 삭제할 때 사용 (디렉터리 내부의 하위 파일 및 폴더까지 일괄 삭제).

### 파일 링크 (`ln`)
기존 파일에 새로운 이름을 부여하여 접근하는 방법입니다.
- **하드 링크 (Hard Link):** `ln [원본] [링크명]`
  - 원본 파일과 동일한 **inode 번호**를 공유함.
  - 하나를 수정하면 양쪽 모두 수정 내용이 반영되며, 원본을 삭제해도 링크 파일은 그대로 남아 데이터에 접근할 수 있음.
- **심볼릭 링크 (Symbolic Link):** `ln -s [원본] [링크명]`
  - 윈도우의 '바로가기'와 같은 개념으로 원본의 **경로**를 가리킴.
  - 원본과 링크의 inode 번호가 서로 다름.
  - 원본 파일을 삭제하면 심볼릭 링크는 연결이 끊어져(깨져서) 사용할 수 없음.

## 2. 리눅스의 문서 편집기 (4장 기초)
### 편집기의 종류
- **GUI 기반:** gedit (윈도우의 메모장과 유사).
- **터미널 기반 (화면 편집기):** 
  - **nano / pico:** 초보자가 사용하기 쉬운 비모드형 편집기.
  - **emacs:** 전문가용으로 강력하지만 단축키가 복잡함.
  - **vi (vim):** 리눅스의 가장 대표적인 표준 화면 편집기. 입력 모드와 명령 모드가 구분된 **모드형 편집기**.

## 3. vi (vim) 편집기 사용법
vi는 마우스를 사용하지 않고 키보드만으로 모든 문서 작업을 처리합니다.

### 모드 전환
- **명령 모드 (초기 화면):** 커서 이동, 복사, 삭제, 붙여넣기 등 수행.
- **입력 모드 전환 (명령 -> 입력):**
  - `i`: 커서 현재 위치부터 입력.
  - `a`: 커서 다음 위치부터 입력.
  - `o`: 커서 아래에 새로운 빈 행을 만들고 입력.
  - `O`: 커서 위에 새로운 빈 행을 만들고 입력.
  - 입력 모드에서 빠져나올 때는 `Esc` 키를 누름.
- **마지막 행 모드 (저장 및 종료):** 명령 모드에서 `:`(콜론)을 입력하여 진입.
  - `:q` (종료), `:q!` (저장하지 않고 강제 종료).
  - `:w` (저장), `:wq` 또는 `ZZ` (저장하고 종료).

### 커서 이동 (명령 모드)
- **방향 이동:** `h`(좌), `j`(하), `k`(상), `l`(우).
- **행 내 이동:** `0` 또는 `^`(행의 처음), `$`(행의 마지막).
- **단어 이동:** `w`(다음 단어 첫 글자), `b`(이전 단어 첫 글자), `e`(현재/다음 단어의 마지막 글자).
- **전체 화면 이동:** `Ctrl + F`(다음 화면), `G`(파일의 맨 끝), `:숫자`(해당 행 번호로 이동).

### 내용 수정 및 삭제
- **수정:** `r`(커서 위치의 한 글자 수정), `cw`(단어 수정).
- **삭제:** 
  - `x`: 커서 위치의 한 글자 삭제.
  - `dd`: 커서가 있는 현재 행 전체 삭제(잘라내기).
- **명령 취소:** `u` (실행 취소, Undo).

### 복사, 잘라내기, 붙여넣기
- **복사 및 붙여넣기:** 
  - `yy`: 현재 행 복사 (예: `3yy`는 3줄 복사).
  - `p`: 커서 아래(또는 다음 위치)에 붙여넣기.
  - `P`: 커서 위(또는 이전 위치)에 붙여넣기.
- **네임드 버퍼:** `"a yy` (a라는 이름의 버퍼에 복사) -> `"a p` (a 버퍼의 내용 붙여넣기).

### 검색 및 바꾸기 (마지막 행 모드)
- **검색:** `/문자열` (아래로 검색), `?문자열` (위로 검색). `n`키를 눌러 다음 검색 결과로 이동.
- **치환(바꾸기):** `:[범위]s/[기존문자]/[새문자]/[옵션]`
  - 예: `:s/Linux/Hanbit/` (현재 행에서 첫 번째 일치 항목 변경)
  - 예: `:5,6s/Hanbit/Linux/g` (5~6행에서 일치하는 모든 항목 변경)

### 기타 유용한 명령
- `:r [파일명]`: 다른 파일의 내용을 현재 커서 위치로 불러옴.
- `:! [셸 명령]`: vi를 종료하지 않고 외부 리눅스 셸 명령을 잠시 실행 (`Enter` 누르면 복귀).
- `J`: 현재 행과 다음 행을 이어 붙임(행 연결).
