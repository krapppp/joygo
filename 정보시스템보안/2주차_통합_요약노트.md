# 📑 정보시스템보안 2주차 통합 요약 노트

## 1. 리눅스 데스크톱 환경 소개
### X 윈도(X Window System)와 웨일랜드(Wayland)
- **X 윈도:** 네트워크 기반의 서버/클라이언트 구조를 가진 리눅스의 전통적인 그래픽 시스템. GUI 구성 요소의 통일된 규칙을 강제하지 않아 다양한 윈도 매니저 개발이 가능함.
- **웨일랜드:** X 윈도의 복잡성을 보완한 간소화된 최신 윈도 시스템. 최신 우분투 및 GNOME에서 기본 윈도 시스템으로 채택됨.

### 주요 데스크톱 환경 (Desktop Environment)
- **GNOME (그놈):** 우분투 24.04의 기본 데스크톱 환경 (GNOME 46). 직관적이고 윈도와 유사한 사용자 경험 제공.
- **기타 환경:** KDE, Xfce, 판테온 등 (터미널에서 `sudo apt install` 명령으로 손쉽게 다른 데스크톱 환경으로 변경 가능).

---

## 2. 우분투 데스크톱 환경 사용
- **현재 활동 및 작업 표시 독(Dock):** 실행 중인 앱과 자주 사용하는 앱 표시. (윈도의 작업 표시줄 역할)
- **앱 센터:** 소프트웨어를 검색하고 GUI 환경에서 손쉽게 설치할 수 있는 도구.
- **기본 앱:** 파일 관리자, 파이어폭스(웹 브라우저), 텍스트 편집기, 이미지 보기, 시스템 설정 등.

## 3. 리눅스 터미널(CLI) 및 기초 명령어
### 터미널 접속 및 종료
- **실행:** 우분투 환경에서 [앱 표시] -> [터미널] 실행.
- **원격 접속(SSH):** 윈도우에서 `PuTTY` 등의 프로그램을 사용하여 리눅스의 IP 주소로 원격 접속 가능 (단, 리눅스에 SSH 서버가 설치되어 있어야 함).
- **종료:** `exit` 명령어 입력 또는 `Ctrl + D` 입력.

### 프롬프트와 단축키
- **프롬프트 형태:** 
  - 일반 사용자: `$` (예: `user1@hanbit:~$`)
  - 최고 관리자(root): `#`
- **단축키:** 
  - `Ctrl + W`: 단어 지우기
  - `Ctrl + U`: 현재 입력 중인 행 전체 지우기

### 명령어 구조와 기초 명령
- **구조:** `명령어 [옵션] [인자]` (예: `ls -a /tmp`)
- **man (Manual):** 명령어의 사용 설명서를 출력하는 핵심 명령어.
  - 사용법: `man [명령어]` (예: `man ls`)
  - 내부 조작: 스페이스바(다음 장), `h`(도움말), `q`(종료)
- **passwd:** 로그인한 계정의 비밀번호를 변경하는 명령어. (로그인 ID는 최고 관리자만 변경 가능).
- **clear:** 터미널 화면을 깨끗하게 지우는 명령어.
