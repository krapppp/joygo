# 📑 웹프로그래밍 6주차 통합 요약 노트

## 1. 리스트 만들기
HTML에서는 세 가지 종류의 리스트 태그를 제공하며, 각 항목은 `<li>` 태그로 작성합니다. (단, 정의 리스트 제외)
- **`<ol>` (순서 있는 리스트):** 1, 2, 3.. 또는 A, B, C.. 등 순번(마커)이 붙는 리스트. `type` 속성으로 마커 변경 가능.
- **`<ul>` (순서 없는 리스트):** 동그라미(블릿) 마커가 붙는 리스트.
- **`<dl>` (정의 리스트):** 용어와 그에 대한 설명을 나열할 때 사용. 
  - `<dt>`: 용어 (Term)
  - `<dd>`: 설명 (Description)

## 2. 표 만들기 (`<table>`)
데이터를 행(Row)과 열(Column)로 구성된 그리드 형태로 출력합니다.
- **기본 구조:**
  - `<table>`: 표 전체를 감싸는 컨테이너. (테두리 속성: `border="1"`)
  - `<caption>`: 표의 제목 (반드시 table 직후에 와야 함).
  - `<thead>`, `<tbody>`, `<tfoot>`: 표를 머리글, 본문, 바닥글 구역으로 그룹화.
- **행과 셀 생성:**
  - `<tr>` (Table Row): 하나의 가로 행을 만듦.
  - `<th>` (Table Heading): 제목 셀. 글자가 굵게 표시되고 가운데 정렬됨.
  - `<td>` (Table Data): 일반 데이터 셀. 내부에 텍스트뿐만 아니라 `<img>` 등 다른 태그도 삽입 가능.

## 3. 웹 폼 (Web Form) 기초
사용자로부터 데이터를 입력받아 웹 서버로 전송하기 위한 양식입니다. (로그인, 검색창 등)
- **`<form>` 태그:** 입력 요소들을 감싸는 폼 영역.
  - `name`: 폼의 식별 이름.
  - `action`: 제출(Submit) 버튼을 눌렀을 때, 데이터를 처리할 웹 서버의 프로그램 주소.
  - `method`: 서버로 데이터를 전송하는 방식 (`GET` 또는 `POST`).
- **기본 입력 요소:**
  - `<input type="text">`: 한 줄짜리 일반 텍스트 입력 창.
  - `<input type="password">`: 비밀번호 입력 창 (문자가 `*` 등으로 가려짐).
  - `<textarea>`: 여러 줄을 입력할 수 있는 텍스트 상자. (`rows`, `cols` 속성으로 크기 지정).
