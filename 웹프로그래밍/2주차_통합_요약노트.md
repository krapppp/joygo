# 📑 웹프로그래밍 2주차 통합 요약 노트

## 1. HTML5 페이지 기본 구조
### 필수 태그 및 지시어
- **`<!DOCTYPE html>`:** 현재 문서가 HTML5 표준 문서임을 웹 브라우저에 알리는 지시어.
- **`<html>`:** 문서의 최상위 루트 태그.
- **`<head>`:** 문서의 메타데이터(문서 제목, 자바스크립트 코드, CSS 스타일 정의 등)가 들어가는 공간.
- **`<body>`:** 웹 브라우저 화면에 직접 출력되는 본문 콘텐츠(텍스트, 이미지, 테이블, 동영상 등)가 들어가는 공간.

---

## 2. HTML 태그의 주요 특징
- **태그 구성:**
  - 시작태그와 종료태그 쌍으로 이루어지는 형태: `<html>...</html>`, `<title>문서의 제목</title>`
  - 종료태그가 없는 단독 태그: `<br>`, `<hr>`
- **대소문자 무관:** 태그와 속성 이름은 대소문자를 구분하지 않음 (`<HTML>` = `<html>`).
- **여백 처리:** 속성값 내의 불필요한 공백은 HTML5 표준에 어긋남.
- **주석:** `<!-- 주석 내용 -->` (브라우저 화면에 출력되지 않음).

---

## 3. 핵심 기본 태그 및 예제 학습
### 문서 및 문단 관련 태그
- **`<title>`:** 웹 브라우저의 타이틀바(또는 탭)에 보여질 문서 제목을 설정함.
  - *툴팁 생성:* `title` 속성을 사용하면 마우스를 올렸을 때 툴팁이 나타나도록 지정할 수 있음. 예: `<h1 title="설명">`
- **`<h1>` ~ `<h6>`:** 문단의 제목을 나타내는 헤딩 태그. 숫자가 작을수록 텍스트의 크기가 크고 강조됨.
- **`<p>`:** 본문을 여러 단락으로 나눌 때 사용.
  - 코드 상에서 여러 개의 빈칸이나 엔터키는 웹 브라우저에서 모두 **하나의 공백(빈칸)**으로 취급됨.
- **`<br>`:** 새로운 줄로 넘어가기 (줄바꿈). 여러 번 쓰면 쓴 횟수만큼 줄이 넘어감.
- **`<hr>`:** 수평선을 그어 단락이나 주제를 시각적으로 분리.
- **`<pre>`:** 개발자가 입력한 포맷(빈칸, 엔터키 등 줄바꿈과 띄어쓰기)을 그대로 화면에 출력.

### 텍스트 꾸미기 태그
- **진하게:** `<b>`, `<strong>` (strong은 의미론적으로 '중요함'을 강조)
- **이탤릭(기울임):** `<i>`, `<em>` (em은 의미론적인 강조)
- **기타 텍스트 서식:**
  - `<small>`: 한 단계 작은 문자
  - `<del>`: 삭제선
  - `<ins>`: 추가 (밑줄)
  - `<sup>`: 윗첨자 / `<sub>`: 아래첨자
  - `<mark>`: 하이라이팅(형광펜 효과)

## 4. 문자, 기호, 심볼 입력 (엔터티/코드)
HTML5의 문자는 유니코드 문자셋과 UTF-8 코드 체계를 따름. 키보드로 입력이 어렵거나 HTML의 예약어(태그 기호)와 충돌하는 기호는 특수 엔터티 표현이나 코드로 작성해야 함.
- **`<`:** `&lt;` 또는 `&#60;`
- **`>`:** `&gt;` 또는 `&#62;`
- **`&` (앰퍼샌드):** `&amp;` 또는 `&#38;`
- **`"` (따옴표):** `&quot;` 또는 `&#34;`
- **빈칸(공백):** `&nbsp;` 또는 `&#160;`
- **기타 기호:** `&copy;`(©), `&divide;`(÷), `&radic;`(√)

---

## 5. 블록 태그 vs 인라인 태그
- **블록 태그 (Block Tag)**
  - **특징:** 항상 새로운 라인에서 시작하며, 양옆에 다른 콘텐츠를 배치하지 않고 **한 라인을 독점**함.
  - **사례:** `<div>`, `<p>`, `<h1>~<h6>`, `<ul>`, `<hr>`
- **인라인 태그 (Inline Tag)**
  - **특징:** 블록 속에 삽입되어 **블록의 일부**로 출력되며, 줄이 바뀌지 않음.
  - **사례:** `<span>`, `<strong>`, `<a>`, `<img>`, `<b>`

*대표적으로 `<div>`는 여러 요소를 묶어 블록 단위로 스타일(CSS)을 지정할 때, `<span>`은 줄바꿈 없이 텍스트 일부 영역의 스타일만 변경할 때 자주 사용됨.*
