# 📑 웹프로그래밍 7주차 통합 요약 노트

## 1. 데이터 목록과 버튼 요소
### 드롭다운 목록 (`<datalist>`)
- `<input type="text">`에 사용자가 입력할 때 미리 정의된 선택지(자동완성 목록)를 제공하는 기능입니다.
- `<input>`의 `list` 속성과 `<datalist>`의 `id`를 연결하여 사용합니다. 항목은 `<option>`으로 정의합니다.

### 다양한 버튼 태그
- **`<input>` 활용:** `<input type="button">`, `<input type="reset">`(초기화), `<input type="submit">`(데이터 폼 전송), `<input type="image">`(이미지를 클릭 버튼으로 사용).
- **`<button>` 태그:** 닫는 태그가 있으며 태그 내부에 텍스트나 이미지를 자유롭게 삽입할 수 있어 더욱 유연함. (예: `<button type="button"><img src=".."> 버튼</button>`)

## 2. 선택형 입력 폼
- **체크박스 (`<input type="checkbox">`):** 다중 선택이 가능한 입력 박스. `checked` 속성을 넣으면 기본으로 체크된 상태가 됨.
- **라디오 버튼 (`<input type="radio">`):** 그룹 내에서 단 하나만 선택 가능한 버튼. 서로 같은 그룹으로 묶기 위해서는 반드시 **`name` 속성 값을 동일하게** 맞추어야 함.
- **콤보 박스 (`<select>`):** 클릭 시 드롭다운으로 펼쳐지는 목록. `<option>` 태그로 항목을 정의하며, 기본 선택을 지정하려면 `selected` 속성을 사용함.

## 3. 캡션 지정 및 기타 입력 폼
### `<label>` (폼 요소 캡션)
- 사용자가 체크박스나 라디오버튼의 텍스트(캡션)만 클릭해도 폼 요소가 선택되도록 클릭 영역을 넓혀주는 매우 유용한 태그.
- **방식 1:** `<label>` 텍스트와 `<input>` `</label>` 로 아예 감싸기.
- **방식 2:** `<label for="id이름">`과 `<input id="id이름">`을 서로 연결하기.

### 색상 및 시간 정보 입력 (HTML5)
- **색상 입력 (`type="color"`):** 컬러 다이얼로그(팔레트)를 띄워 색상 코드(`#rrggbb`)를 입력받음.
- **시간/날짜 입력:** 브라우저 기본 달력 및 시계 인터페이스를 띄움.
  - `type="date"`: 연-월-일
  - `type="time"`: 시:분
  - `type="month"`, `type="week"`, `type="datetime-local"` 등.
