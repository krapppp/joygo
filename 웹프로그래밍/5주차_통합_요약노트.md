# 📑 웹프로그래밍 5주차 통합 요약 노트

## 1. 메타 데이터 태그
메타 데이터란 데이터를 설명하는 데이터로, 웹 페이지의 보이지 않는 정보를 설정합니다. 메타 태그들은 주로 `<head>` 태그 내부에 작성됩니다.

- **`<base>`:** 웹 페이지 내의 모든 상대 경로가 기준이 될 기본 URL을 지정합니다. (예: `<base href="http://www.mysite.com/">`)
- **`<link>`:** 외부 자원을 문서에 연결할 때 사용합니다. (예: CSS 스타일시트 연결 - `<link rel="stylesheet" href="style.css">`)
- **`<meta>`:** 문서의 저작자, 문자 인코딩(charset), 검색 엔진을 위한 키워드 등을 정의합니다.
  - 문자 인코딩: `<meta charset="utf-8">`
  - 설명/키워드: `<meta name="description" content="...">`

## 2. 이미지와 인라인 프레임
### 이미지 삽입 (`<img>`)
- `src` 속성에 이미지 경로를, `alt` 속성에 이미지를 띄울 수 없을 때 보여줄 대체 텍스트를 지정합니다.
- 예: `<img src="media/photo.jpg" width="150" height="200" alt="사진">`

### 인라인 프레임 (`<iframe>`)
- 현재 HTML 페이지 안에 또 다른 HTML 페이지를 창(프레임) 형태로 삽입합니다.
- **주요 속성:** `src` (불러올 문서 경로), `srcdoc` (iframe 안에 직접 HTML 코드를 작성할 때 사용 - src를 무시함), `name` (프레임 윈도우의 이름).
- **`target` 속성:** 링크(`<a>`)를 클릭했을 때 결과가 열릴 윈도우를 지정합니다.
  - `_blank`(새 창), `_self`(현재 창), `_parent`(부모 창), `_top`(최상위 브라우저 창), `프레임이름`(특정 iframe 창).

## 3. 미디어 삽입 (`<audio>`, `<video>`)
HTML5부터는 별도의 플러그인(플래시 등) 없이 브라우저 자체적으로 오디오와 비디오를 재생할 수 있는 표준 태그를 지원합니다.
- **`<video>` 태그:** `src`, `width`, `height`, `controls`(제어 버튼 출력), `autoplay`(자동 재생), `loop`(반복 재생) 속성 제공.
- **`<audio>` 태그:** `src`, `controls`, `autoplay`, `loop` 속성 제공.
- *참고:* `<source>` 태그를 하위 요소로 사용하여 브라우저가 지원하는 코덱에 맞춰 재생할 수 있도록 여러 포맷의 파일을 제공할 수 있습니다.
