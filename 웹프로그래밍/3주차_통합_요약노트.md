# 📑 웹프로그래밍 3주차 통합 요약 노트

> **참고:** 3주차 강의는 주로 2주차에 학습한 HTML5의 기본 개념과 태그를 반복/심화 학습하는 내용으로 구성되어 있습니다.

## 1. HTML 문서 구조 복습
- **필수 뼈대:** `<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`
- 문서의 타이틀바(탭) 이름은 `<head>` 내의 `<title>` 태그로 지정하며, 태그에 `title="설명"` 속성을 주면 마우스 오버 시 **툴팁**이 나타납니다.

---

## 2. 문단 및 서식 제어 태그
- **`<h1>` ~ `<h6>`:** 제목(Heading)을 나타내며, h1이 가장 큽니다.
- **`<p>`:** 단락(Paragraph)을 나눕니다.
- **`<br>` / `<hr>`:** 줄바꿈(Break)과 수평선(Horizontal Rule)을 삽입하는 단독 태그입니다.
- **`<pre>`:** 코드에 작성된 띄어쓰기와 엔터키(줄바꿈) 등의 포맷을 브라우저에 그대로 출력합니다.

---

## 3. 텍스트 강조 및 기호 삽입
- **텍스트 꾸미기:** 
  - 진하게: `<b>`, `<strong>`
  - 기울임: `<i>`, `<em>`
  - 하이라이트: `<mark>`
  - 첨자: `<sup>`(위), `<sub>`(아래)
- **HTML 엔터티 기호:**
  - 공백: `&nbsp;` (`&#160;`)
  - `<` / `>`: `&lt;` / `&gt;`
  - `"` (따옴표): `&quot;`

---

## 4. 구조화를 위한 태그 구분 (핵심)
HTML 태그는 화면을 차지하는 방식에 따라 크게 두 가지로 나뉩니다.

1. **블록 태그 (Block Tag)**
   - **특징:** 항상 새로운 라인에서 시작하며, 부모 요소의 전체 너비를 차지하여 **한 라인을 독점**합니다. 양옆에 다른 콘텐츠가 올 수 없습니다.
   - **종류:** `<div>` (가장 대표적), `<p>`, `<h1>`~`<h6>`, `<ul>` 등.

2. **인라인 태그 (Inline Tag)**
   - **특징:** 새로운 줄로 넘어가지 않고, 텍스트 흐름에 맞춰 **블록의 일부 영역**만 차지합니다.
   - **종류:** `<span>` (가장 대표적), `<strong>`, `<a>`, `<img>` 등.

*요약: `<div>`는 영역 자체의 배경색 등 큰 덩어리의 레이아웃을 잡을 때 사용하고, `<span>`은 단어 단위로 글자색을 변경하는 등 텍스트 중간에 스타일을 입힐 때 사용합니다.*
