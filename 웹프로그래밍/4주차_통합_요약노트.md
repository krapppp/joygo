# 📑 웹프로그래밍 4주차 통합 요약 노트

## 1. HTML5 문서 구조화와 시맨틱 웹
### 시맨틱 웹(Semantic Web)이란?
- **개념:** 웹 문서에 시맨틱 태그를 사용하여 문서의 구조와 의미를 명확히 전달하는 웹.
- **필요성:** 
    - **검색 엔진 최적화(SEO):** 구글 등 검색 엔진이 페이지의 핵심 내용을 파악하기 쉬움.
    - **웹 접근성:** 시각 장애인용 스크린 리더 등이 문서를 정확히 해석 가능.
    - **유지보수:** 개발자가 소스 코드만 보고도 해당 영역의 역할을 즉시 파악 가능.

### 주요 시맨틱 구조 태그
- **`<header>`:** 페이지나 섹션의 머리말 (제목, 로고 등).
- **`<nav>`:** 네비게이션 링크들을 모아놓은 섹션 (메뉴, 목차 등).
- **`<section>`:** 문서의 장(Chapter)이나 절을 구성하는 일반적인 영역.
- **`<article>`:** 본문과 독립적으로 구성 가능한 콘텐츠 (블로그 포스트, 뉴스 기사 등).
- **`<aside>`:** 본문 옆의 보조 내용 (광고, 관련 링크, 팁 등).
- **`<footer>`:** 페이지 하단 정보 (저작권, 저자 정보, 연락처 등).

## 2. 시맨틱 블록 및 인라인 태그
### 시맨틱 블록 태그
- **`<figure>` / `<figcaption>`:** 사진, 도표, 소스 코드 등을 그룹화하고 설명(캡션)을 붙임.
- **`<details>` / `<summary>`:** 상세 정보를 숨겨두었다가 사용자가 제목(`summary`)을 클릭하면 내용을 펼쳐서 보여줌 (Q&A 등에 활용).

### 시맨틱 인라인 태그
- **`<mark>`:** 중요한 텍스트 하이라이팅 (노란색 배경).
- **`<time>`:** 시간 정보임을 명시.
- **`<meter>`:** 주어진 범위 내의 데이터 양 표시 (예: 하드디스크 용량).
- **`<progress>`:** 작업 진행 정도 표시 (예: 다운로드 바).

## 3. 하이퍼링크 (Hyperlink) 상세
### `<a>` 태그의 핵심 속성
- **`href`:** 연결할 URL 주소 또는 파일 경로.
- **`target`:** 링크를 어디에 열지 결정.
    - `_blank`: 새 탭(새 창)에서 열기.
    - `_self`: 현재 창에서 열기 (기본값).
- **텍스트 및 이미지 링크:** 글자뿐만 아니라 `<img>` 태그를 `<a>`로 감싸서 이미지 클릭 시 이동하게 할 수 있음.

### 링크의 텍스트 색상 (기본값)
- **Blue:** 방문 전 링크.
- **Purple:** 방문 후 링크.
- **Red:** 마우스로 누르고 있는 동안.
*(CSS를 통해 변경 가능)*

## 4. 앵커(Anchor) 만들기
- **개념:** 동일한 페이지 내의 특정 위치로 순식간에 이동하는 기능.
- **방법:**
    1. 이동하고 싶은 목적지 요소에 `id` 속성을 부여함. 
       - 예: `<h3 id="intro">서론</h3>`
    2. 클릭할 링크의 `href`에 `#id이름`을 작성함.
       - 예: `<a href="#intro">서론으로 이동</a>`
- **활용:** 페이지 하단에서 '맨 위로 가기' 버튼을 만들거나, 긴 문서의 목차를 만들 때 필수적임.
