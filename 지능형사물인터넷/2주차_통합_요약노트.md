# 📑 지능형 사물 인터넷 2주차 통합 요약 노트

## 1. 아두이노 스케치 구조와 기초 문법
### 스케치(코드)의 기본 구조
아두이노 프로그램은 내부적으로 `main()` 함수가 숨겨져 작동하며, 사용자는 주로 아래 두 함수를 작성합니다.
- `setup()`: 보드에 전원이 들어오거나 리셋될 때 **처음에 한 번만 실행**되는 함수. (주로 핀 모드 설정, 시리얼 통신 초기화 등을 담당)
- `loop()`: `setup()`이 끝난 후 전원이 꺼질 때까지 **무한 반복 실행**되는 함수. (메인 동작 논리 포함)

### 함수와 변수
- **변수(Variable):** 데이터를 저장하는 공간. 자료형에 따라 정수형(`int`), 실수형(`float`) 등으로 나뉨.
- **함수(Function):** 특정 작업을 수행하는 코드의 집합.
  - 반복되는 코드를 하나로 묶어 코드의 재사용성을 높이고 복잡한 기능을 쉽게 구현할 수 있도록 도와줌.
  - 예시: `int f(int x) { return x + 1; }` (정수 x를 받아 1을 더해 반환하는 함수)

## 2. 시리얼 통신 (아두이노의 '입')
아두이노 보드와 컴퓨터(PC) 간에 데이터를 주고받아, 센서 값이나 작동 상태를 화면(시리얼 모니터)에 출력하는 기능입니다.

### Serial.println()
- **초기화:** `setup()` 함수 내에 `Serial.begin(115200);`를 작성하여 통신 속도(보드레이트)를 설정함.
- **기본 출력:** 텍스트, 정수, 실수, 문자 등을 한 줄씩 출력함.
  - `Serial.println("Hello PC");`
  - `Serial.println(1.23456);`
- **형식 변환 출력:** 진수 변환이나 소수점 자릿수 조절이 가능함.
  - `Serial.println(78, DEC);` (10진수)
  - `Serial.println(78, HEX);` (16진수)
  - `Serial.println(78, BIN);` (2진수)
  - `Serial.println(1.23456, 2);` (소수점 둘째 자리까지 출력)

## 3. 디지털 출력과 LED 제어 (아두이노의 '윙크')
아두이노의 디지털 핀을 제어하여 LED를 켜거나 끄는 기초 회로 실습입니다.

### 핵심 함수
- **pinMode(핀번호, 모드):** 특정 핀을 입력(INPUT)으로 쓸지 출력(OUTPUT)으로 쓸지 설정.
  - 예: `pinMode(13, OUTPUT);` (13번 핀을 출력용으로 설정)
- **digitalWrite(핀번호, 상태):** 출력으로 설정된 핀에 전기를 보낼지(HIGH) 끊을지(LOW) 결정.
  - **LED 켜기:** `digitalWrite(13, HIGH);` (5V 전압 출력)
  - **LED 끄기:** `digitalWrite(13, LOW);` (0V 전압 출력)

### 확장 개념 (빛의 연주)
- **analogWrite():** 단순히 켜고 끄는(HIGH/LOW) 것을 넘어, PWM 핀(`~` 표시 핀)을 이용해 LED의 밝기를 미세하게 조절할 수 있는 기능.
