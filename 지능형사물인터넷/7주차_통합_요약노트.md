# 📑 지능형 사물 인터넷 7주차 통합 요약 노트

## 1. 아두이노의 귀: 시리얼 통신 입력 (`Serial.read`)
PC의 시리얼 모니터에서 사용자가 키보드로 입력한 값을 아두이노가 읽어들여 동작(LED 제어 등)을 결정하는 기능입니다.

### 핵심 시리얼(Serial) 함수
- **`Serial.begin(115200)`:** 통신 속도 설정 (`setup()`에 작성).
- **`Serial.available()`:** 수신된(도착한) 데이터가 있는지 확인하여, 데이터가 있으면 0보다 큰 값을 반환. 보통 `if(Serial.available() > 0)` 형태로 사용.
- **`Serial.read()`:** 수신 버퍼에서 데이터를 **1바이트(하나의 문자)**씩 읽어옴. (예: `char userInput = Serial.read();`)

### 응용: `switch-case` 문 활용
읽어들인 문자에 따라 다중 분기 처리를 할 때 `switch` 문이 유용합니다.
```cpp
switch (userInput) {
  case 'n': digitalWrite(LED, HIGH); break;  // 'n' 입력 시 LED 켜기
  case 'f': digitalWrite(LED, LOW); break;   // 'f' 입력 시 LED 끄기
  default: break;  // 예외 처리
}
```

## 2. 아두이노의 촉각: 디지털 입력 (`digitalRead`)
푸시 버튼(스위치)이나 각종 디지털 센서가 눌렸는지 떼어졌는지를 감지하는 입력 기능입니다.

### `digitalRead`의 구조
- **핀 설정:** 입력으로 사용할 핀은 반드시 `setup()`에서 `pinMode(핀번호, INPUT);`으로 설정해야 함.
- **상태 읽기:** `int val = digitalRead(핀번호);`
  - 스위치가 눌려 전류가 통하면 `HIGH` (1) 반환.
  - 스위치가 떼어져 전류가 끊기면 `LOW` (0) 반환.
  *(단, 풀업/풀다운 저항 회로 구성 방식에 따라 눌렸을 때 LOW가 될 수도 있으므로 주의)*

### 활용 사례: LED 주사위 만들기
- 여러 개의 LED 핀 번호를 배열로 묶어둠 (`int led[6] = {3, 5, 6, 9, 10, 11};`).
- **버튼을 누르고 있는 동안 (`if (buttonInput == HIGH)`)**:
  - `for`문을 이용해 6개의 LED를 매우 빠른 속도로 번갈아 가며 깜빡이게 만듦 (룰렛 효과).
- **버튼을 떼는 순간 (`if (buttonInput == LOW) break;`)**:
  - `break`를 통해 반복문을 즉시 탈출하여, 가장 마지막에 켜져 있던 LED 하나만 켜진 상태로 멈추게 됨 (마치 주사위 결과가 나온 것과 같음).
