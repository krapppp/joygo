# 📑 지능형 사물 인터넷 6주차 통합 요약 노트

## 1. 아날로그 출력 (analogWrite)
디지털 핀의 HIGH(5V), LOW(0V) 단순 출력을 넘어, **PWM(Pulse Width Modulation, 펄스 폭 변조)** 기술을 이용해 평균 전압을 제어하여 LED 밝기나 모터 속도를 미세하게 조절하는 기술입니다.

### PWM (Pulse Width Modulation)
- **개념:** 아주 짧은 시간(약 1/500초) 단위로 전기를 켰다 껐다를 반복하며, 켜져 있는 시간(Duty Cycle)의 비율을 조절해 평균 전압을 낮추거나 높이는 기술. 눈의 착시를 이용함.
- **지원 핀:** 아두이노 보드에서 핀 번호 앞에 물결표(`~`)가 있는 핀만 사용 가능 (예: D3, D5, D6, D9, D10, D11).
- **사용법:** `analogWrite(핀번호, 0~255 값)`
  - `0`: 0% Duty Cycle (완전 꺼짐, 0V)
  - `127`: 50% Duty Cycle (중간 밝기, 평균 2.5V)
  - `255`: 100% Duty Cycle (완전 켜짐, 5V)

### 빛의 연주 응용
- `for` 반복문과 `analogWrite`를 결합하여, 밝기 값(`t_high`)을 0에서 255까지 순차적으로 증가시키면 LED가 서서히 밝아지는 페이드 인(Fade-in) 효과를 구현할 수 있음.

## 2. 네오픽셀 (NeoPixel) 제어
일반 LED와 달리 내부에 통신 칩이 내장되어 있어, **단 하나의 데이터 선(Data 핀)**만으로 수십~수백 개의 RGB LED의 색상과 밝기를 개별적으로 제어할 수 있는 스마트 LED입니다.

### Adafruit_NeoPixel 라이브러리 기초
외부 라이브러리인 `<Adafruit_NeoPixel.h>`를 먼저 설치하고 인클루드해야 합니다.
- **객체 생성:** `Adafruit_NeoPixel Pixels(LED개수, 핀번호, 옵션);`
- **주요 함수:**
  - `Pixels.begin()`: 네오픽셀 제어 시작 및 초기화.
  - `Pixels.setBrightness(0~255)`: 전체 네오픽셀의 최대 밝기 설정.
  - `Pixels.Color(R, G, B)`: 0~255 사이의 Red, Green, Blue 값을 조합하여 고유한 색상 데이터 생성.
  - `Pixels.setPixelColor(인덱스, 색상)`: 0부터 시작하는 `인덱스` 번째 LED에 색상을 지정 (메모리에만 기록됨).
  - `Pixels.fill(색상, 시작위치, 개수)`: 여러 개의 LED를 한 번에 같은 색으로 채움.
  - **`Pixels.show()`:** 메모리에 설정된 색상 값들을 **실제 LED 하드웨어로 전송하여 불을 켬** (가장 중요).
  - `Pixels.clear()`: 모든 LED 색상을 0(꺼짐)으로 초기화함 (단, `show()`를 호출해야 실제로 꺼짐).

### 배열(Array)과 네오픽셀의 활용
- 다양한 색상(`uint32_t`)을 배열에 저장해두고, `for`문을 이용해 각 LED 인덱스나 행(Row)별로 무지개색을 순차적으로 점등하는 화려한 효과를 손쉽게 구현할 수 있음.
