# 📑 지능형 사물 인터넷 4주차 통합 요약 노트

## 1. 제어문 (조건문과 반복문)
아두이노 스케치(C/C++)에서 프로그램의 흐름을 제어하는 핵심 문법입니다.

### 조건문 (if, else if, else)
- 특정 조건이 참(True)일 때만 지정된 명령을 실행합니다.
- 센서 값에 따라 특정한 동작(예: LED 켜기/끄기)을 수행할 때 주로 사용합니다.
- **형식:** `if (조건식) { 참일 때 실행; } else { 거짓일 때 실행; }`

### 반복문 (for, while)
- **for문:** 반복 횟수가 정해져 있거나 일정한 규칙에 따라 순차적으로 반복할 때 사용합니다.
  - `for(int i = 0; i <= 3; i++) { 실행문; }` (0부터 3까지 4번 반복)
- **while문:** 반복 횟수가 정해져 있지 않고, 특정 조건이 만족되는 동안 계속 반복할 때 사용합니다.
  - 불규칙한 반복이나 `while(true)` 형태의 무한 루프 구현에 유용하며, `break` 문과 함께 사용하여 루프를 탈출할 수 있습니다.

## 2. 배열 (Array)과 응용
동일한 자료형의 데이터를 여러 개 저장할 때 사용하며, 핀 번호 관리에 매우 유용합니다.

### 배열의 선언과 초기화
- `int arr[5] = {10, 20, 30};` (나머지 2개는 0으로 자동 초기화됨)
- `int arr[] = {100, 200, 300};` (초기값의 개수에 따라 배열 크기가 자동으로 결정됨)
- 다중 LED를 제어할 때 각 LED의 핀 번호를 배열에 저장하면 `for`문과 결합하여 코드를 획기적으로 줄일 수 있습니다.
  - 예: `int led[8] = {2, 3, 4, 5, 6, 7, 8, 9};` 선언 후 `for`문을 이용해 일괄 `pinMode` 설정 및 제어.

### sizeof() 함수
- 배열이나 변수의 **메모리 크기(Byte)**를 반환합니다.
- 배열의 전체 요소 개수(길이)를 구할 때 유용합니다.
  - `int length = sizeof(arr) / sizeof(arr[0]);`

## 3. 전자 회로 기초 (LED와 브레드보드)
### 브레드보드 (빵판)
- 납땜 없이 전자 부품을 꽂아서 임시 회로를 구성할 수 있는 도구.
- 전원선(+, -)은 가로로 길게 연결되어 있고, 부품을 꽂는 중앙 영역은 세로로 짧게 연결되어 있는 구조를 이해해야 합니다.

### 저항 읽는 법과 LED 연결
- **저항:** 과도한 전류가 흘러 LED가 타는 것을 방지하기 위해 필수적으로 연결해야 합니다. (저항 표면의 띠 색상으로 저항값을 읽음).
- **LED 극성:** 다리가 긴 쪽이 `+` (애노드), 짧은 쪽이 `-` (캐소드)입니다.
- 아두이노의 디지털 핀에서 나온 전원은 저항을 거쳐 LED의 `+` 극으로 들어가고, LED의 `-` 극은 아두이노의 `GND`(그라운드)로 연결되어야 회로가 완성됩니다.
