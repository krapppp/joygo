# 📑 정보시스템기반기술 1주차(1장) 통합 요약 노트

## 1. 운영체제의 개념과 역할
- **운영체제(OS):** 하드웨어와 소프트웨어가 유기적으로 결합하여 작동하도록 돕고, 컴퓨터 자원을 효율적으로 관리하는 '관리자' 역할의 시스템 소프트웨어.
- **역할 및 목표:**
  - 자원 배분 및 관리 (CPU, 메모리, 입출력 장치 등)
  - 자원 보호 (무분별한 접근 차단)
  - 사용자 및 하드웨어 인터페이스 제공

## 2. 컴퓨터 및 운영체제의 발전 과정
- **초기:** 에니악(하드 와이어링 방식) -> 폰노이만 구조 (명령어와 데이터를 메모리에 저장 후 순차 처리, 현대 컴퓨터의 기본 구조).
- **발전 단계:**
  1. **일괄 작업 시스템:** 한 번에 하나씩 순차적으로 처리.
  2. **대화형 시스템:** 키보드/모니터 등장으로 실시간 상호작용 가능.
  3. **시분할 시스템:** 짧은 시간 단위(타임 슬라이스)로 작업을 분할하여 동시에 실행되는 것처럼 보이는 멀티프로그래밍 구현.
  4. **분산 시스템:** 클라이언트-서버, P2P, 클라우드 컴퓨팅, IoT 등으로 발전.

## 3. 운영체제의 구성
운영체제는 크게 **사용자 인터페이스**와 **커널(Kernel)**로 나뉩니다.
- **커널(Kernel):** 프로세스, 메모리, 저장장치 등을 관리하는 핵심 중앙 제어 소프트웨어.
- **시스템 호출 (System Call):** 사용자나 응용 프로그램이 커널의 자원을 사용하기 위해 거쳐야 하는 공식적인 요청 창구 (사용자 모드 <-> 커널 모드 전환).
- **디바이스 드라이버:** 커널이 주변 하드웨어 장치와 데이터를 주고받기 위한 인터페이스.

## 4. 운영체제의 종류
- **유닉스 계열:** C 언어로 개발되어 이식성이 뛰어나며 소스 코드를 공개하여 발전 (BSD 유닉스 등).
- **리눅스:** 개인용 컴퓨터에서 동작하는 유닉스 호환 커널. GNU 프로젝트(자유소프트웨어)와 GPL 라이선스를 따름.
- **개인용 컴퓨터 OS:** 애플의 맥OS (GUI 적극 도입), 마이크로소프트의 MS-DOS 및 윈도우.
- **모바일 OS:** iOS (BSD 유닉스 기반), 안드로이드 (리눅스 커널 기반의 오픈 소스).
