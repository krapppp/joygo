## 1. 인공지능, 머신러닝, 딥러닝의 관계
### 3가지 개념의 포함 관계 (인공지능 > 머신러닝 > 딥러닝)
- **인공지능 (AI):** 인간이 가진 지적 능력을 컴퓨터로 구현하는 모든 기술 (강인공지능, 약인공지능).
- **머신러닝 (Machine Learning):** 알고리즘을 이용해 데이터를 분석하고, 컴퓨터가 스스로 학습하여 새로운 규칙을 발견하고 판단이나 예측을 하도록 만드는 기술.
- **딥러닝 (Deep Learning):** 여러 은닉층(Layer)을 깊게(Deep) 쌓은 '심층 인공신경망(ANN)'을 사용하여 머신러닝 학습을 수행하는 기술.

### 머신러닝과 딥러닝의 주요 차이점
1. **인간의 개입 유무:** 머신러닝은 사람이 학습 데이터에 정답을 주거나 **특징을 추출(Feature Extraction)**하는 개입이 필요하지만, 딥러닝은 스스로 특징을 추출하고 학습함.
2. **데이터 의존도:** 딥러닝은 엄청난 양의 데이터가 주어져야만 좋은 성능을 발휘하지만, 충분한 데이터가 있으면 사람이 인지하지 못한 특징까지 찾아낼 수 있음.
3. **심층신경망 사용 여부:** 딥러닝만이 심층신경망 구조를 사용하여 복잡한 문제를 해결함 (블랙박스 형태).

---

## 2. 머신러닝을 사용하는 이유
- **기존 프로그래밍(규칙 기반)의 한계:** 대용량 데이터와 많은 변수가 존재하거나 상황이 계속 변할 때(예: 주식 예측, 얼굴 인식, 사기 탐지 등), 사람이 모든 `IF-THEN` 규칙을 수동으로 입력하는 것은 불가능함.
- **유용성:** 머신러닝은 데이터를 기반으로 스스로 패턴을 찾아내므로 복잡하고 변화무쌍한 문제를 해결하는 데 매우 적합함.

---

## 3. 머신러닝의 3대 분류
1. **지도학습 (Supervised Learning):** 문제(입력 데이터, x)와 정답(레이블, y)을 함께 제공하여 학습시킴. 새로운 문제에 대한 올바른 답을 찾아내는 **'분류(Classification)'**나 **'예측(Regression)'**에 사용.
2. **비지도학습 (Unsupervised Learning):** 정답(레이블) 없이 데이터만 주고 컴퓨터 스스로 데이터 간의 규칙성과 관계를 찾아내게 함. 비슷한 특성의 데이터를 묶는 **'군집(Clustering)'**에 주로 사용.
3. **강화학습 (Reinforcement Learning):** 정해진 환경 속에서 에이전트(주체)가 어떤 행동을 취할 때마다 **보상(Reward) 또는 벌**을 주어, 가장 보상이 큰 최적의 행동 패턴을 스스로 찾아가게 하는 학습 방법 (예: 알파고, 게임 AI).

---

## 4. 지도학습: 분류 (Classification) 알고리즘
레이블이 있는 데이터를 학습하여, 새로 입력된 데이터가 어느 그룹에 속하는지 식별하는 기법입니다. (이진 분류, 다중 분류)

### 대표적인 분류 알고리즘
1. **K-최근접 이웃 (KNN, K-Nearest Neighbors):** 
   - 새로운 데이터 주변에 있는 K개의 기존 데이터들이 어느 그룹에 속해 있는지를 보고(다수결 원칙) 자신의 그룹을 결정하는 알고리즘. (K값 설정이 중요함).
2. **서포트 벡터 머신 (SVM, Support Vector Machine):**
   - 두 분류 사이의 여백인 **마진(Margin)**을 극대화하는 최적의 경계선(결정 경계)을 그어 데이터를 분류하는 방법. 정확도가 높으나 학습 시간이 오래 걸림.
3. **의사결정나무 (Decision Tree):**
   - 스무고개처럼 상위 노드부터 분류 기준값에 따라 하위 노드로 뻗어 나가며 데이터를 분류하는 직관적인 모델. 분석 과정을 눈으로 볼 수 있어 **설명 가능성(White Box)**이 매우 뛰어남.
4. **로지스틱 회귀 (Logistic Regression):**
   - 데이터가 특정 범주에 속할 **확률을 0~1 사이의 값**으로 계산한 뒤, 기준선(예: 0.5)을 넘는지에 따라 그룹을 분류하는 알고리즘. (연속형 변수를 예측하는 선형 회귀와 달리 이산형/이분 변수 분류에 주로 쓰임).
