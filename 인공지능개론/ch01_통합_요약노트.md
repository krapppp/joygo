# 📑 인공지능개론 ch01 통합 요약 노트

## 1. 인공지능(AI)의 이해
### 개념 및 정의
- **지능(Intelligence):** 이해하고 배우는 능력으로, 생각하고 이해함으로써 행동하는 인간 고유의 성질.
- **인공지능(AI):** 컴퓨터가 학습하고 생각하여 스스로 판단할 수 있도록 만드는 기술.
- **프로그래밍 vs AI:**
  - *프로그래밍:* 사람이 모든 케이스를 직접 구현.
  - *AI:* 컴퓨터 스스로 데이터를 학습하여 판단하고 행동.

### 목표와 시작
- **다트머스 컨퍼런스(1956):** 인공지능이라는 용어가 처음 등장한 학술회의.
- **전제:** 지능의 모든 특성이 정확히 기술될 수 있어 기계가 이를 모사할 수 있다는 가정.
- **최종 목표:** 인간의 다양한 능력을 컴퓨터가 대신하게 하고, 궁극적으로 로봇의 지능을 향상시키는 것.

## 2. 인공지능의 순기능과 역기능
### 순기능
1. **노동 대체:** 정신적·육체적 노동 대체 (예: 아마존 키바 로봇, 로봇 바리스타).
2. **자동화(RPA):** 단순 반복 업무의 소프트웨어 로봇화 (예: 보험금 청구 심사).
3. **개인 능력 격차 완화:** 비전문가도 전문가 수준의 업무 수행 지원 (예: 로보 어드바이저).

### 역기능
1. **멍청한 AI (Dumb AI):** 잘못된 데이터 학습 시 오판 가능성 (예: 챗GPT의 환각 현상).
2. **책임 소재:** 사고 발생 시 법적·윤리적 책임 주체 모호 (예: 자율주행차 사고).
3. **경제적 불평등:** 일자리 대체로 인한 생계 위협 및 빈부격차 심화.
4. **예측 불가능성:** 학습되지 않은 변수 발생 시 오작동 (예: 이세돌 vs 알파고).

## 3. 인공지능의 연대기 (라이프 사이클)
인공지능은 태동기, 암흑기, 성장기, 성숙기를 거치며 발전해왔습니다.

- **1차 태동기 (1950~1956):** 앨런 튜링의 **튜링 테스트**(기계의 지능 판단) 등장.
- **2차 태동기 (1956~1974):** 다트머스 컨퍼런스, AI 연구 방향 정립.
- **1차 암흑기 (1974~1980):** 낮은 연구 성과로 인한 투자 감소.
- **성장기 (1980~1987):** **전문가 시스템**(전문 지식을 컴퓨터에 저장) 유행.
- **2차 암흑기 (1987~1993):** 전문가 시스템의 유지비 문제와 한계 봉착.
- **1차 성숙기 (1993~2000):** **인공신경망** 연구, 머신러닝/딥러닝의 기반 마련.
- **2차 성숙기 (2000~2010):** 머신러닝과 딥러닝의 본격적 발전.
- **3차 성숙기 (2010~현재):** **알파고**의 등장, 데이터 기반 자율 학습의 시대.

## 4. 인공지능의 분류
### 지적 수준에 따른 분류
1. **약인공지능(Weak AI):** 특정 분야의 일만 수행 (예: 알파고, 자율주행차).
2. **강인공지능(Strong AI):** 모든 분야에서 범용적으로 사용, 자아 인식 가능.
3. **초인공지능(Super AI):** 인간의 지능을 모든 면에서 압도하는 존재 (**특이점** 시점).

### 구현 및 기능에 따른 분류
- **단순 제어:** 세탁기, 청소기 등 기초 단계.
- **고전적 AI:** 정해진 규칙(패턴)에 따라 행동.
- **머신러닝(ML):** 데이터를 분석하여 학습하고 판단. (특징 추출에 인간 개입)
- **딥러닝(DL):** 기계가 스스로 특징을 추출하고 해답을 찾는 수준.

---

## 5. 인공지능의 한계와 과제
- **인간 의존성:** 머신러닝의 상당 부분은 여전히 인간의 데이터 분류 수작업에 의존함.
- **데이터 편향성:** 편향된 데이터로 학습할 경우 인종차별이나 편견을 형성할 수 있음 (예: MS 챗봇 '테이').
- **설명 가능성:** AI가 내린 결론의 과정을 인간이 이해하기 어려운 '블랙박스' 문제.
