# 📑 직업기초윤리(엑셀) ch03 통합 요약 노트

## 1. 수식 입력의 기초
- **수식의 시작:** 모든 수식은 반드시 등호(`=`)로 시작함.
- **구성 요소:** 연산자(+, -, *, / 등)와 피연산자(숫자, 문자, 셀 주소 등)로 구성됨.
- **참조의 장점:** 값을 직접 입력하는 대신 셀 주소(A1, B1 등)를 사용하면 원본 데이터 변경 시 결과값이 자동으로 갱신됨.
- **다중 셀 입력:** 범위를 선택한 후 수식을 입력하고 `Ctrl` + `Enter`를 누르면 선택한 모든 셀에 동일한 수식이 한꺼번에 적용됨.

## 2. 자동 합계 및 기본 함수
- **자동 합계 도구:** [수식] 탭 - [자동 합계] 기능을 통해 자주 쓰이는 함수를 빠르게 입력 가능.
  - `SUM()`: 합계
  - `AVERAGE()`: 평균
  - `COUNT()`: 숫자가 포함된 셀의 개수
  - `MAX()` / `MIN()`: 최대값 / 최소값
- **자릿수 조절:** 계산 결과의 소수점을 조절하려면 [홈] 탭 - [표시 형식]의 [자릿수 늘림/줄임] 버튼 활용.

## 3. 셀 참조 방식 (가장 중요)
수식을 복사(채우기 핸들 드래그)할 때 셀 주소가 어떻게 변하는지에 따라 구분됨.
- **상대 참조 (A1):** 복사되는 위치에 따라 행과 열 주소가 자동으로 변경됨.
- **절대 참조 ($A$1):** 수식을 어디로 복사하든 참조하는 셀 주소가 **고정**됨. (단축키 `F4`를 눌러 설정)
  - 예: 환율, 할인율 등 모든 계산에서 공통으로 참조해야 하는 단일 셀에 사용.
- **혼합 참조 ($A1 또는 A$1):** 행이나 열 중 하나만 고정함.
  - `$A1`: 열 고정 (가로로 복사해도 A열 유지)
  - `A$1`: 행 고정 (세로로 복사해도 1행 유지)

## 4. 이름 정의하기
- **개념:** 특정 셀이나 범위를 주소(A1:B10) 대신 의미 있는 이름(예: '환율', '할인율')으로 지정하는 기능.
- **장점:** 
  - 수식의 가독성이 높아짐 (예: `=C4/환율`).
  - 이름을 사용하면 자동으로 **절대 참조**처럼 작동하여 수식 복사 시 주소가 변하지 않음.
- **방법:**
  - [이름 상자]에 직접 입력하거나, [수식] 탭 - [이름 정의] 활용.
  - **선택 영역에서 만들기:** 표의 첫 행이나 왼쪽 열의 텍스트를 이용해 여러 범위를 한꺼번에 이름으로 정의 가능.
- **관리:** [수식] 탭 - [이름 관리자]에서 정의된 이름을 수정하거나 삭제할 수 있음.
