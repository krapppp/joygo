# 📑 직업기초윤리(엑셀) ch05 통합 요약 노트

## 1. 함수의 기본 이해
- **함수의 정의:** 특정 결과값을 구하기 위해 수식을 이름으로 미리 정의해 둔 기능.
- **기본 형식:** `=함수명(인수1, 인수2, ...)`
  - 함수에 따라 필요한 인수의 개수와 데이터 형식이 다름.
- **함수 입력 방법:** 
  - 직접 입력하거나 [수식] 탭 - [함수 라이브러리] 그룹의 도구 활용.

## 2. 수학/삼각 및 통계 함수
실무 및 성적 계산 등에서 가장 많이 사용되는 함수들입니다.

### 주요 함수 목록
- **SUM(범위):** 선택한 범위의 합계를 계산함.
- **AVERAGE(범위):** 선택한 범위의 평균을 계산함.
- **COUNT(범위):** 선택한 범위에서 **숫자**가 포함된 셀의 개수만 셈.
- **MAX(범위) / MIN(범위):** 범위 내에서 가장 큰 값 또는 가장 작은 값을 찾음.
- **STDEV.S(범위):** 표본의 표준편차를 계산함 (데이터의 분산 정도 확인).

### 고급 계산 함수
- **SUMPRODUCT(범위1, 범위2):** 대응하는 배열의 원소끼리 곱한 후 그 합계를 구함.
  - 예: `점수 * 가중치`의 합계를 구할 때 유용 (`=SUMPRODUCT(F4:J4, $F$3:$J$3)`).
- **QUOTIENT(피제수, 제수):** 나눗셈의 몫을 구함. (예: `=QUOTIENT(C4, 12)`).
- **ROUND(값, 자릿수):** 숫자를 지정한 자릿수에서 반올림함.
  - 예: `=ROUND(STDEV.S(점수1), 1)` -> 표준편차를 구해 소수점 첫째 자리까지 표시.

## 3. 함수 활용 팁
- **이름 정의와 함수:** 특정 범위를 이름(예: '점수1')으로 정의해 두면 수식 작성이 훨씬 간편해짐 (`=STDEV.S(점수1)`).
- **함수 중첩:** 하나의 수식 안에 여러 함수를 함께 사용 가능 (`ROUND` 함수 안에 `STDEV.S`를 넣는 방식 등).
- **채우기 핸들:** 수식을 하나 작성한 후 채우기 핸들을 드래그하거나 더블클릭하여 나머지 셀에 수식을 빠르게 복사할 수 있음. 이때 참조 방식(상대/절대)에 주의해야 함.
