# 📑 직업기초윤리(엑셀) ch05 통합 요약 노트 (기본 및 고급 함수)

## 1. 함수의 기본 이해
- **함수의 정의:** 특정 결과값을 구하기 위해 수식을 이름으로 미리 정의해 둔 기능.
- **기본 형식:** `=함수명(인수1, 인수2, ...)`
  - 함수에 따라 필요한 인수의 개수와 데이터 형식이 다름.
- **함수 입력 방법:** 
  - 직접 입력하거나 [수식] 탭 - [함수 라이브러리] 그룹의 도구 활용.

---

## 2. 수학/삼각 및 통계 함수 (기본)
실무 및 성적 계산 등에서 가장 많이 사용되는 함수들입니다.
- **SUM(범위):** 선택한 범위의 합계를 계산함.
- **AVERAGE(범위):** 선택한 범위의 평균을 계산함.
- **COUNT(범위):** 선택한 범위에서 **숫자**가 포함된 셀의 개수만 셈.
- **MAX(범위) / MIN(범위):** 범위 내에서 가장 큰 값 또는 가장 작은 값을 찾음.
- **STDEV.S(범위):** 표본의 표준편차를 계산함.
- **SUMPRODUCT(범위1, 범위2):** 대응하는 배열의 원소끼리 곱한 후 그 합계를 구함 (예: 가중치 계산).
- **ROUND(값, 자릿수):** 숫자를 지정한 자릿수에서 반올림함.

---

## 3. 텍스트 함수
문자열의 일부를 추출하거나 합치는 기능입니다.
- **CONCAT(텍스트1, 텍스트2, ...):** 여러 텍스트 문자열을 하나로 연결함. (예: `=CONCAT(A3, "-", B3)`)
- **LEFT(텍스트, 개수):** 문자열의 왼쪽부터 지정한 개수만큼 문자를 추출함. (예: 주민번호 앞 6자리 추출 `=LEFT(E3, 6)`)
- **MID(텍스트, 시작위치, 개수):** 문자열의 지정한 위치부터 원하는 개수만큼 문자를 추출함. (예: 주민번호 8번째 자리 성별 코드 1개 추출 `=MID(E3, 8, 1)`)
- **FIND(찾을문자, 텍스트):** 특정 문자가 시작되는 위치(숫자)를 찾아 반환함. `LEFT` 등과 혼합하여 특정 기호 앞까지의 글자만 추출할 때 유용함.

## 4. 날짜 및 시간 함수
엑셀에서 날짜는 1900년 1월 1일을 기준(숫자 1)으로 하루마다 1씩 증가하는 정수형 숫자로 저장되어 계산됩니다.
- **TODAY():** 시스템의 오늘 날짜를 반환함. (인수 없음)
- **DATE(년, 월, 일):** 각각의 숫자를 조합하여 유효한 날짜 데이터로 변환함.
- **DATEDIF(시작일, 종료일, "단위"):** 두 날짜 사이의 기간(나이, 근속연수 등)을 계산함.
  - `"y"`: 전체 연도 차이
  - `"m"`: 전체 개월 차이
  - `"ym"`: 연도를 제외한 나머지 개월 수 차이 (예: 1년 3개월 차이라면 3을 반환)
  - 응용: `=DATEDIF(L3, TODAY(), "y") & "년 " & DATEDIF(L3, TODAY(), "ym") & "개월"` -> "~년 ~개월" 형태로 출력.

## 5. 논리 및 조건 함수 (고급)
특정 조건을 만족하는지에 따라 다른 결과값을 반환합니다.
- **IF(조건식, 참일때값, 거짓일때값):** 조건이 맞으면 참 값을, 틀리면 거짓 값을 반환.
  - 예: `=IF(E3>=30, "상", "하")`
- **다중 IF (함수 중첩):** 조건이 여러 개일 때 IF 함수 안에 또 IF 함수를 넣어 사용함.
  - 예: `=IF(E3>=34, "상", IF(E3>=28, "중", "하"))`
- **ISODD(숫자):** 숫자가 홀수이면 TRUE를 반환함. (예: 주민번호 성별 코드가 홀수이면 "남", 짝수이면 "여"로 판별할 때 `=IF(ISODD(MID(A3,8,1)), "남", "여")`로 응용 가능).

## 6. 오류 검사 함수
수식 계산 중 에러(`#DIV/0!`, `#N/A` 등)가 발생했을 때 보기 싫은 에러 창 대신 지정한 텍스트나 숫자(0)를 띄워줍니다.
- **IFERROR(수식, 오류시표시할값):** 앞의 수식이 정상적으로 계산되면 그 값을 출력하고, 오류가 발생하면 지정한 값을 출력함.
  - 예: `=IFERROR(AVERAGE(C3:D3), 0)` -> 평균 계산 시 나눌 값이 없어 에러가 나면 0으로 깔끔하게 표시.
