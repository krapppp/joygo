# 📑 직업기초윤리(엑셀) ch01 통합 요약 노트

## 1. 엑셀(Excel) 개요 및 특징
- **기본 개념:** 셀(Cell)이라는 데이터 입력 공간이 모여 워크시트(Worksheet)를 이루는 스프레드시트 프로그램.
- **규모:** 하나의 워크시트는 약 104만 개의 행과 1.6만 개의 열로 구성되며, 파일 하나에 최대 255개의 워크시트 생성 가능.
- **엑셀 2019 주요 새 기능:**
    - **새로운 함수:** `CONCAT`, `IFS`, `MAXIFS`, `MINIFS`, `SWITCH`, `TEXTJOIN` 추가.
    - **시각화 도구:** 지도 차트, 깔때기형 차트, SVG 이미지(아이콘) 및 3D 모델 삽입 기능.
    - **잉크 기능:** 터치 장치에서 펜 두께/색상 설정 및 필기 기능 개선.
    - **기타:** 셀 선택 취소(`Ctrl` 누르고 클릭), 피벗 테이블 기능 향상 등.

---

## 2. 엑셀 기본 화면 구성
1. **빠른 실행 도구 모음:** 자주 사용하는 명령(저장, 실행 취소 등)을 모아둔 곳 (사용자 지정 가능).
2. **제목 표시줄:** 현재 작업 중인 파일 이름 표시.
3. **리본 메뉴:** 기능별로 분류된 탭(홈, 삽입, 수식 등)과 명령 아이콘 모음.
4. **이름 상자:** 현재 선택된 셀의 주소(예: A1)나 정의된 이름을 표시.
5. **수식 입력줄:** 선택한 셀의 실제 내용이나 수식을 입력 및 수정하는 곳.
6. **작업 영역(워크시트):** 실제 데이터를 입력하는 셀들의 집합.
7. **상태 표시줄:** 작업 상태, 시트 보기 형식, 확대/축소 슬라이더 포함.

---

## 3. 파일 시작 및 관리
### 서식 파일 활용
- 엑셀에서 제공하는 디자인 서식(예: 예산서, 일정표 등)을 검색하여 즉시 사용 가능.
- 사용자가 직접 만든 서식을 등록하여 재사용할 수도 있음.

### 저장 및 보안
- **파일 저장:** [파일] - [정보] 또는 [다른 이름으로 저장] 활용.
- **암호 설정:** 
    - **파일 암호:** [정보] - [통합 문서 보호] - [암호 설정]을 통해 파일 열기 시 암호 요구.
    - **일반 옵션:** 저장 대화상자의 [도구] - [일반 옵션]에서 열기/쓰기 암호를 각각 설정 가능.

### 인쇄 설정
- **인쇄 영역:** 특정 범위만 드래그하여 [페이지 레이아웃] - [인쇄 영역 설정] 가능.
- **페이지 설정:** 여백 조절, 페이지 가운데 맞춤(가로/세로), 머리글/바닥글(페이지 번호, 날짜 등) 편집.
- **인쇄 제목:** [시트] 탭에서 '반복할 행'을 선택하면 모든 페이지 상단에 제목 행이 출력됨.

---

## 4. 워크시트(Worksheet) 다루기
- **시트 관리:** 시트 탭에서 우클릭을 통해 삽입, 삭제, 이름 바꾸기, 이동/복사 가능.
- **탭 색상:** 시트별로 탭의 색상을 지정하여 시각적으로 구분 가능.
- **시트 및 문서 보호:**
    - **시트 보호:** 특정 시트의 셀 내용 수정을 방지.
    - **통합 문서 보호:** 시트의 이동, 삭제, 이름 변경 등 구조적인 변경을 방지.
