# 📑 직업기초윤리(엑셀) ch02 통합 요약 노트

## 1. 셀(Cell) 선택과 편집
### 셀 선택 및 이동
- **단일/연속 셀 선택:** 마우스 드래그 또는 `Shift` + 방향키.
- **비연속 셀 선택:** `Ctrl` 키를 누른 상태에서 마우스로 클릭 또는 드래그.
- **특정 조건 셀(빈 셀 등) 선택:** [홈] 탭 - [편집] 그룹 - [찾기 및 선택] - [이동 옵션] 활용 (예: 빈 셀만 일괄 선택 가능).
- **키보드 이동 단축키:**
  - `Enter`: 아래로 이동 / `Shift` + `Enter`: 위로 이동
  - `Tab`: 오른쪽으로 이동 / `Shift` + `Tab`: 왼쪽으로 이동
  - `Ctrl` + `Home`: 워크시트의 첫 번째 셀(A1)로 이동
  - `Ctrl` + `방향키`: 데이터가 입력된 범위의 맨 끝(상하좌우) 셀로 이동

### 데이터 복사 및 셀 병합
- **복사/붙여넣기:** `Ctrl` + `C` / `Ctrl` + `V` (선택된 셀 테두리가 점선으로 표시됨).
- **자동 완성:** 동일한 열에 이미 입력된 데이터의 첫 글자를 입력하면 나머지 글자가 자동으로 표시됨 (이때 `Enter`를 누르면 자동 완성됨).
- **셀 병합:** 범위 선택 후 [홈] 탭 - [병합하고 가운데 맞춤].
  - *전체 병합:* 행 단위로 병합.
  - *셀 병합:* 하나의 통짜 셀로 병합(가운데 맞춤 없음).
  - 병합된 셀을 다시 누르면 분할됨.

## 2. 행/열 다루기
### 삽입 및 삭제
- **방법:** 행/열 머리글(알파벳 또는 숫자)을 우클릭하여 삽입 또는 삭제.
- **다중 삽입:** 여러 개의 행/열 머리글을 드래그하여 선택한 후 삽입하면, 선택한 개수만큼 한 번에 빈 행/열이 삽입됨.
- **복사 삽입:** 열(또는 행)을 복사(`Ctrl`+`C`)한 후, 삽입할 위치의 머리글에서 [복사한 셀 삽입]을 누르면 원본 데이터가 통째로 삽입됨.

### 숨기기 기능
- 특정 행이나 열 머리글을 우클릭하여 **[숨기기]** 처리 가능.
- **숨기기 취소:** 숨겨진 열의 양옆 머리글을 드래그하여 선택한 후 우클릭 - [숨기기 취소]. (예: D열이 숨겨진 경우 C열과 E열을 함께 선택 후 취소).

## 3. 데이터 입력하기
### 문자 데이터 입력
- 텍스트를 입력하고 `Enter`를 누르면 아래 셀로 이동하면서 계속 입력 가능.
- **열 너비 조절:** 데이터가 셀 너비보다 길어 가려질 경우, 열 머리글 사이의 경계선을 마우스로 드래그(또는 더블클릭)하여 너비를 넓힘.
- **셀 내 줄바꿈 (매우 중요):** 하나의 셀 안에서 두 줄 이상으로 데이터를 입력하려면 **`Alt` + `Enter`** 키를 사용함.
