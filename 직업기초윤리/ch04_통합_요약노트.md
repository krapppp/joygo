## 1. 표시 형식 (Cell Formatting)
### 기본 표시 형식
- **쉼표 스타일(`,`)**: [홈] 탭 - [표시 형식] 그룹에서 설정하며, 숫자에 천 단위 구분 기호를 추가함.
- **백분율 스타일(`%`)**: 소수 데이터를 백분율 형식으로 변환함.
- **회계와 통화**: 
  - *회계:* 기호(￦)가 왼쪽 끝에 고정됨.
  - *통화:* 기호(￦)가 숫자 바로 앞에 붙음.
- **날짜/시간:** 연, 월, 일 형식을 원하는 대로 변경 가능.

### 사용자 지정 표시 형식 (단축키: `Ctrl` + `1`)
- **개념:** 기존 형식을 직접 수정하여 단위(권, 개, 원 등)나 색상을 입히는 기능.
- **형식 코드:** 
  - `0`: 숫자가 없어도 0을 표시함 (예: `000` -> 7이면 `007`로 표시).
  - `#`: 숫자가 없으면 생략함 (예: `#,###`).
  - `@`: 문자 데이터가 표시될 위치 지정 (예: `@님`).
  - `G/표준`: 입력된 데이터를 있는 그대로 표시.
- **작성 사례:** 
  - `G/표준"권"`: 숫자 뒤에 '권'이 붙음.
  - `#,##0,"천원"`: 천 단위에서 절삭하고 '천원' 단위를 붙임.
  - `000-000-0000`: 숫자를 전화번호/학번 형식으로 자동 변환.

## 2. 조건부 서식 (Conditional Formatting)
### 규칙 기반 서식
특정한 조건을 만족하는 셀만 강조하거나 시각적 도구를 추가하는 기능.
- **셀 강조 규칙:** [홈] 탭 - [스타일] - [조건부 서식] - [같음] / [보다 큼] / [텍스트 포함] 등을 통해 특정 셀의 색을 변경함.
- **상위/하위 규칙:** 상위 10% 또는 하위 3개 등 상위권/하위권 데이터만 골라 서식 적용 가능.
- **데이터 시각화 도구:** 
  - **데이터 막대:** 셀 값의 크기에 비례하여 막대 그래프가 셀 안에 그려짐.
  - **아이콘 집합:** 방향(화살표), 도형(신호등) 등으로 값의 추이를 시각화함.

### 수식을 이용한 조건부 서식 (매우 중요)
단순한 값 비교가 아니라, **행 전체**에 서식을 적용하고 싶을 때 사용함.
- **방법:** [새 규칙] - [수식을 사용하여 서식을 지정할 셀 결정] 선택.
- **수식 작성 예:** `=$E3 >= 20000` 
  - 열 주소 앞에 `$`를 붙여 **혼합 참조**로 작성해야 해당 행의 모든 셀이 같은 조건(E열 값)을 기준으로 서식이 적용됨.
- **규칙 관리:** [조건부 서식] - [규칙 관리]에서 기존 규칙의 우선순위를 정하거나 조건을 수정/삭제할 수 있음.
