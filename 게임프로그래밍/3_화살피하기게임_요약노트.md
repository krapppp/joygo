# 🐱 게임프로그래밍: 고양이 화살 피하기 게임 (Cat Escape)

랜덤한 위치에서 떨어지는 화살을 피하고, 충돌 시 체력 게이지가 깎이는 게임 로직입니다.

## 💡 주요 학습 내용
- **연속 입력 감지:** `Input.GetKey`를 이용한 실시간 캐릭터 이동 제어.
- **프리팹(Prefab):** 동일한 오브젝트를 대량 생산하기 위한 템플릿 사용.
- **Instantiate:** `Instantiate(arrowPrefab)`를 통한 동적 오브젝트 생성.
- **델타 시간:** `Time.deltaTime`을 이용한 기기 독립적 타이머 구현.
- **UI 제어:** `Image.fillAmount`를 이용한 HP 바 시각화.

## 📄 핵심 스크립트 1: PlayerController.cs (좌우 이동)
```csharp
using UnityEngine;

public class PlayerController : MonoBehaviour
{
    void Update()
    {
        // 방향키 입력에 따라 캐릭터 좌표 이동
        if(Input.GetKey(KeyCode.LeftArrow)) {
            transform.Translate(-0.25f, 0.0f, 0.0f);
        }
        if(Input.GetKey(KeyCode.RightArrow)) {
            transform.Translate(0.25f, 0.0f, 0.0f);
        }
    }
}
```

## 📄 핵심 스크립트 2: ArrowGenerator.cs (화살 생성기)
```csharp
using UnityEngine;

public class ArrowGenerator : MonoBehaviour
{
    public GameObject arrowPrefab; // 화살 프리팹 연결
    float span = 1.0f;  // 생성 주기 (1초)
    float delta = 0.0f; // 경과 시간 체크용

    void Update()
    {
        this.delta += Time.deltaTime; // 프레임 간 경과 시간 합산
        if(this.delta > this.span)
        {
            this.delta = 0.0f;
            // 화살 생성 및 랜덤 위치 지정 (-6 ~ 6)
            GameObject go = Instantiate(arrowPrefab);
            int px = Random.Range(-6, 7);
            go.transform.position = new Vector3(px, 7, 0);
        }
    }
}
```

## 📄 핵심 스크립트 3: GameDirector.cs (HP 관리)
```csharp
using UnityEngine;
using UnityEngine.UI; // UI 시스템 사용 시 필수

public class GameDirector : MonoBehaviour
{
    GameObject hpGauge;

    void Start() {
        // UI 오브젝트 탐색
        this.hpGauge = GameObject.Find("hpGauge");
    }

    public void DecreaseHp() {
        // 이미지 채우기 비율을 10%씩 감소
        this.hpGauge.GetComponent<Image>().fillAmount -= 0.1f;
    }
}
```
