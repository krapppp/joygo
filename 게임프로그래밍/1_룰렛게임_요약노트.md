# 🎡 게임프로그래밍: 룰렛 게임 (Roulette Game)

마우스 클릭 시 룰렛이 회전하고, 마찰력에 의해 서서히 멈추는 로직을 구현한 프로젝트입니다.

## 💡 주요 학습 내용
- **유니티 라이프사이클:** `Start()`(초기화)와 `Update()`(매 프레임 실행)의 차이.
- **마우스 입력:** `Input.GetMouseButtonDown(0)`을 이용한 클릭 감지.
- **회전 제어:** `transform.Rotate`를 이용한 오브젝트 회전.
- **감속 구현:** 매 프레임마다 속도에 실수(0.96f)를 곱하여 자연스러운 멈춤 구현.

## 📄 핵심 스크립트: RouletteController.cs
```csharp
using UnityEngine;

public class RouletteController : MonoBehaviour
{
    float rotSpeed = 0.0f;  // 회전 속도

    void Update()
    {
        // 1. 마우스 왼쪽 버튼 클릭 시 회전 속도 부여
        if(Input.GetMouseButtonDown(0))
        {
            rotSpeed = 100.0f;
        }

        // 2. Z축을 기준으로 설정된 속도만큼 회전
        transform.Rotate(0, 0, rotSpeed);

        // 3. 감속 효과 적용 (마찰력)
        rotSpeed *= 0.96f; 
    }
}
```
