# 🚗 게임프로그래밍: 자동차 스와이프 게임 (Car Swipe Game)

마우스 드래그(스와이프)한 거리를 계산하여 자동차를 그 방향과 힘으로 이동시키는 프로젝트입니다.

## 💡 주요 학습 내용
- **입력 좌표 추출:** `Input.mousePosition`을 통해 클릭 시작점과 끝점 저장.
- **벡터 계산:** (끝점 - 시작점) 연산으로 스와이프 길이 및 방향 계산.
- **오브젝트 이동:** `transform.Translate`를 활용한 직선 이동.
- **프레임 동기화:** `Application.targetFrameRate` 설정 이해.

## 📄 핵심 스크립트: CarController.cs
```csharp
using UnityEngine;

public class CarController : MonoBehaviour
{
    float speed = 0.0f;
    Vector2 startPos; // 클릭 시작 위치

    void Start()
    {
        // FPS를 60으로 고정
        Application.targetFrameRate = 60;
    }

    void Update()
    {
        // 1. 클릭 시점의 좌표 저장
        if(Input.GetMouseButtonDown(0))
        {
            startPos = Input.mousePosition;
        }
        // 2. 버튼을 떼는 시점의 좌표로 거리(스와이프 길이) 계산
        else if(Input.GetMouseButtonUp(0))
        {
            Vector2 endPos = Input.mousePosition;
            float swipeLength = endPos.x - startPos.x;

            // 이동 속도 설정 (거리에 비례)
            speed = swipeLength / 500.0f;
        }

        // 3. 자동차 실제 이동 및 감속
        transform.Translate(speed, 0.0f, 0.0f);
        speed *= 0.98f;
    }
}
```
